**********    O que são Gráficos ?    **********

O gráfico é uma representação com forma geométrica (o que significa que ele precisa ser desenhado em uma tela do computador) construída de maneira exata e precisa a partir de informações numéricas (para que o grafico transmita a informação precisa construi-lo de maneira precisa através dos dados) obtidas através de pesquisas e organizadas em uma tabela (pesquisa neste caso é buscar os dados na fonte ou através de pesquisa).



**********    Como a Linguagem R trata as visualizações de Dados ?    **********

Existe um consenso na comunidade de Data Science, que a linguagem R fornece algumas das melhores ferramentas para geração de gráficos

Os gráficos podem ser replicados, modificados e publicados com apenas poucas linhas de código.

O pacote ggplot2 é considerado um dos melhores pacotes para visualização de dados do mercado. (existe a versao para linguagem R e também para Python).

Os plots (gráficos bem simples) são objetos construídos através de funções com atributos (tamanho, cor).

Podemos criar gráficos bem mais completos e profissionais usando pacotes como ggplot2, lattice.

**********    Gramática dos Gráficos    **********

Descrição para cada um dos 7 elementos da Gramática dos Gráficos


Elemento         -         Descrição

Dados            -         Conjunto de dados a ser analisado

Estética         -	   Escala em que mapeamos os dados (escala influencia interpretação dos                                dados)

Geometria        -         Elementos visuais usados para representar os dados (tipo de linha em 			           grafico de linha, tipo de ponto em grafico de dispersão)

Facets           -         Visualização do gráfico em porções menores 

Estatística      -         Representação e análise dos dados (leitura dos dados)

Coordenadas      -         Área na qual o gráfico será construído

Temas            -         Visual geral do gráfico
